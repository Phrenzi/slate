# PatronApp / Boosters

## get all boosters today

```shell
curl "https://phrenzi.com/api/patron_app/establishments/c3d12f1a-1f0e-4ade-ae6a-49660db2b2ea/boosters" \
  -H "Content-Type: application/json" \
  -H "Authorization: token"
```
